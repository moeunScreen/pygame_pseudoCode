# pygame_pseudoCode
Pseudocode based on Making Games with Python & Pygame.

이 저장소는 『Making Games with Python & Pygame』를 학습하며 직접 정리한 내용을 모아둔 저장소입니다.

포함된 내용은 다음과 같습니다.

Pygame에서 자주 사용되는 함수와 기본 문법 및 규칙 정리
각 게임의 동작 흐름을 이해하기 위한 의사코드(Pseudocode) 정리

정리된 의사코드는 다음 게임들을 기반으로 작성되었습니다.

짝 맞추기 게임 (Memory Puzzle)
음을 맞추는 깜빡이는 시뮬레이터 (Simulate)
스네이크 게임 (Wormy)
테트리스
성장 다람쥐 게임 (Squirrel Eat Squirrel)
별 밀기 게임 (Star Pusher)

이 저장소의 목적은 완성된 게임 코드를 제공하는 것이 아니라,게임의 구조와 알고리즘, 프로그램의 흐름을 이해하기 쉽도록 핵심 내용을 정리하고 기록하는 데 있습니다.
